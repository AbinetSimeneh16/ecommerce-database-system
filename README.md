# E-commerce Database System

## Project Overview
This project is a relational database system for an online store.  
It was developed using SQL and PostgreSQL.

The database manages:
- Product categories
- Products
- Customers
- Orders
- Order items

---

## Database Features
- Relational database design
- Primary Keys and Foreign Keys
- Data insertion and management
- SQL queries for filtering, joins, aggregation, and reporting

---

## Tables
1. Categories
2. Products
3. Customers
4. Orders
5. Order_Items
